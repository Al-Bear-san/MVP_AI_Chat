# 🤖 AI-чат-бот для бронирования

Мультиплатформенный AI-чат-бот для бронирования столиков в ресторанах.

## 🚀 Возможности

- ✅ Бронирование через Telegram, ВКонтакте, MAX
- ✅ AI-генерация напоминаний и ответов на FAQ
- ✅ Интеграция с Google Calendar
- ✅ Кросс-канальная идентификация пользователей
- ✅ FSM (конечный автомат) для пошагового бронирования
- ✅ Rate limiting и защита от спама
- ✅ Структурированное логирование
- ✅ Мониторинг через Sentry/Prometheus

## 📋 Требования

- Python 3.11+
- PostgreSQL 15+
- Redis 7+
- Docker & Docker Compose
